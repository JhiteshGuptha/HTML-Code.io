## üìã Topics Covered

### 1. The Foundation
* **HelloWorld.html**: The standard "Hello World" entry point demonstrating the root `<html>` element, `<head>` metadata, and the `<body>` content area.
* **BasicTags.html**: Covers structural essentials, including all six levels of headings (`<h1>`‚Äì`<h6>`), paragraph tags, and the `<pre>` tag for preserving whitespace.
* **Favicon.html**: Demonstrates how to link a brand logo to the browser tab using the `<link rel="icon">` attribute.

### 2. Text Formatting & Semantic Layout
* **TextFormatting.html**: Styles including **Bold**, *Italic*, <u>Underline</u>, `<mark>` for highlighting, and mathematical notation using `<sup>` and `<sub>`.
* **SematicTags.html**: Best practices for modern web structure using meaningful tags like `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>`.
* **NonSematicTags.html**: Explains the difference between **Block-level** elements (`<div>`) and **Inline** elements (`<span>`).
* **MovieSummary.html**: A real-world application of `<big>`, `<small>`, and `<hr>` tags to create a clean content layout.

### 3. Navigation & Multimedia
* **HyperLink.html**: Mastery of the anchor tag (`<a>`) for both **Absolute** (external) and **Relative** (internal) linking.
* **Images.html**: Implementation of images and GIFs, including height/width adjustments and accessibility via `alt` text.
* **ClickableImage.html**: Techniques for nesting images inside anchor tags to create visual buttons and logos.
* **Audio.html** & **Video.html**: Methods for embedding media with native browser controls, autoplay, and looping functionality.
* **IFrame.html**: How to embed external content like YouTube videos or secondary websites directly into your page.

### 4. Data Organization & Forms
* **List.html**: Comprehensive examples of **Unordered** (`<ul>`), **Ordered** (`<ol>`), and **Description** (`<dl>`) lists.
* **Table.html**: Creating data grids using `<table>`, `<tr>`, `<th>`, and `<td>`, featuring border and alignment attributes.
* **Form.html**: An extensive guide to user input, featuring text fields, passwords, radio buttons, checkboxes, file uploads, and dropdown menus.
